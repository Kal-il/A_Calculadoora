# Calculadora Simples

Este é um projeto de uma calculadora simples desenvolvida em HTML, CSS e JavaScript. A calculadora permite realizar operações básicas de adição, subtração, multiplicação, divisão, cálculo de raiz quadrada e potência.

## Funcionalidades

- Cálculo de operações matemáticas básicas (+, -, *, /).
- Cálculo de raiz quadrada (usando o símbolo √ seguido de um número).
- Cálculo de potência (usando a sintaxe 'base ^ potência').

## Como Usar

1. Clone este repositório para o seu ambiente local.
2. Abra o arquivo `index.html` em um navegador da web.
3. Use os botões na calculadora para inserir números e realizar operações.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
